# paw-forest
UL Web technologies II practical assignment

# Main Functionality
The main purpose of this animal adoption service is to:
- publicly and dynamically display all animals available for adoption across all locations
- support user search queries for animals using filtering by attributes
- ensure user registration and login, after which the user can:
    - apply for adoption
    - apply for visitation
    - make donations to the adoption center
- ensure employee login, after which the employee can:
    - view and manage all animals available for adoption
    - view and manage all users
    - view and manage all adoption applications
    - view all donations
    - view and manage all visitation applications
    - view and manage all animal medical prescriptions
